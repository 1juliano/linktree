# Linktree Clone

Um projeto simples de página estilo **Linktree**, desenvolvido para centralizar e compartilhar seus links em um único lugar.

---

## ✨ Funcionalidades

- Página responsiva
- Foto de perfil
- Lista de links personalizados
- Ícones de redes sociais
- Tema moderno e minimalista
- Fácil personalização

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

---
